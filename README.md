## 🌳 VrikshaRakshak – Tree Cutting Permission System|

> 🌱 Enabling transparent and efficient environmental approvals through digital governance.

### 📌 Overview

**VrikshaRakshak** is a government-based web application developed to digitize and manage the **Tree Cutting Permission approval process**.

The system ensures proper monitoring and approval of tree cutting requests during **construction phases (CC & OC)** with a structured multi-level authority workflow.

---

### 🚀 Key Features

* 📄 Online Application Submission (CC & OC)
* 🔁 Multi-Level Authority Approval Workflow
* 🌳 Tree Inspection & Verification Process
* 📊 Dashboard & Status Tracking
* 📥 Final NOC Upload System
* 🔗 API-Based Data Autofetch (BPMS)
* 🔍 Real-Time Application Tracking
* ❌ No Payment Integration (as per project requirement)

---

### 🏗️ Modules

#### 🔹 Commencement Certificate (CC)

* Application Submission
* Site Inspection by Authority
* Multi-Level Review Process
* Final NOC Approval

#### 🔹 Occupancy Certificate (OC)

* Auto-fetch of CC Data (if linked)
* Multi-Level Authority Review
* Final NOC Upload
* Process Completion

---

### ⚙️ Workflow

#### 🔹 CC Flow

1. Login to Portal
2. Select ULB & Project
3. Choose Tree Cutting Permission
4. Fill Application Details
5. Submit Application
6. Site Inspection
7. Multi-Level Authority Review
8. Final Approval & NOC Upload
9. Process Completion

---

#### 🔹 OC Flow

1. Login & Select Project
2. Choose NOC Service
3. Fill Application Details
4. Submit Application
5. Site Inspection
6. Multi-Level Review
7. Final NOC Upload
8. Process Completion
